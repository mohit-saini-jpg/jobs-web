🔍 WHAT'S HAPPENING
───────────────────────────────────────────────────────────────

Original CSS:
  table-layout: fixed;  → Narrows columns too much
  width: 35%;          → Restricts label column
  word-break: break-word; → Breaks at EVERY character

Result: Each character wraps to new line (BROKEN)


Our Fix:
  table-layout: auto;  → Let columns size naturally
  width: auto;         → Allow flexible sizing
  word-break: normal;  → Break only at spaces

Result: Proper multi-line text wrapping (FIXED)
