How to Check MHT CET PCB Response Sheet 2026

The CET Cell has released the PCB response sheet along with the provisional answer key so that students can compare their recorded responses with the official answers. To view the response sheet, candidates must log in through the same portal used for the answer key.

Once logged in, the dashboard shows the question paper, options, candidate’s marked responses, and the correct option IDs for each question. This combined view of answer key and response sheet is essential for accurate score calculation and for raising any objections within the given window.
