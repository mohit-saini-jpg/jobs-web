### ISSUE-002 — Duplicate, oversized JSON fetches per page load · High
- **Status:** verified
- **Commit:** f7b44ed
- **Files:** [script.js](script.js)
- **Fix:** New `getJSON(path)` memoizes JSON fetches. dynamic-sections.json/jobs.json/tools.json/services.json/header_links.json each fetched **1×** per load instead of 2–3×. Removed `cache: "no-store"` so HTTP caching engages on later navigations.
- **Verification:** `performance.getEntriesByType('resource')` shows exactly 5 JSON fetches on home (one per file).

### ISSUE-003 — Tailwind CDN warning on resume-maker.html · Medium
- **Status:** verified
- **Commit:** 5faceed
- **Files:** [resume-maker.html](resume-maker.html), [resume-maker.tailwind.css](resume-maker.tailwind.css), [.tailwind-build/](.tailwind-build/)
- **Fix:** Replaced `https://cdn.tailwindcss.com` with a precompiled, minified Tailwind 3 stylesheet (18 KB) generated by scanning the page's classes plus the existing custom config (brand palette, soft/card shadows, pulseSoft animation). Production warning gone, layout unchanged.
- **Regen:** `cd .tailwind-build && npx tailwindcss@3 -c tailwind.config.js -i input.css -o ../resume-maker.tailwind.css --minify`

### ISSUE-004 — Insecure iframe sandbox combination · Medium
- **Status:** verified
- **Commit:** 37677b2
- **Files:** [view.html](view.html)
- **Fix:** Dropped `allow-same-origin` from the viewer iframe. The combination of `allow-scripts` + `allow-same-origin` lets sandboxed content escape the sandbox; the iframe loads cross-origin URLs which the browser already treats as a unique origin, so the attribute was redundant and unsafe.

### ISSUE-005 — Mixed absolute and relative URLs in primary navigation · Medium
- **Status:** verified
- **Commits:** d0aae0d, 3135d3c
- **Files:** [header.html](header.html), [index.html](index.html), [header_links.json](header_links.json), [script.js](script.js), [dynamic-sections.json](dynamic-sections.json), [jobs.json](jobs.json), [tools.json](tools.json), [services.json](services.json)
- **Fix:** Internal href values like `https://www.topsarkarijobs.com/view.html?...` switched to relative paths so the data travels cleanly across hosts. Canonical, og:url, twitter:image, and JSON-LD URLs intentionally stay absolute (SEO requires it).
