❓ FAQ

Compression ke baad output image-based hota hai, isliye text dikhega lekin select/copy nahi hoga — scanned-jaisa output milta hai.
Nahi, pehle PDF Unlock tool use karein.
150MB+ tak support hai. Bahut badi files mein browser thoda time le sakta hai.
150 DPI (Good) zyadatar documents ke liye best balance hai. Print/scan quality chahiye to 300+ DPI use karein, WhatsApp/email ke liye chhoti size chahiye to 72-96 DPI. Final size hamesha aapke set kiye Target File Size ke hisaab se adjust hoti hai.
Aap 1 KB se 100 MB tak koi bhi target size set kar sakte hain. Tool chuni gayi DPI par pages render karke unhe re-encode karta hai taaki us size ke jitna close pahunchne ki koshish ho sake.
