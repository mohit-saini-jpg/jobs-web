Add Name & Date in Image — Free Online Tool (Sarkari Form ke liye)

Bahut si Sarkari job applications, exam forms aur government document uploads mein zaroori hota hai ki aap apni photo par apna Naam aur Date (D.O.P — Date of Photo) likhein. Ye free online tool aapko bina kisi software ya app ke, seedhe browser mein photo par Naam aur Date add karne deta hai — SSC, UPSC, Railway (RRB), IBPS Bank, State PSC, Police aur Army sabhi form ke liye perfect.

Is tool se aap kya kar sakte hain

  • Naam aur Date likhein — Naam upar, Date niche, dono alag centered lines mein — bilkul form requirement jaisa.
  • Extra white space — niche safed patti add karke usmein Naam & Date print karein (jaise passport photo ke niche hota hai).
  • Font size aur colour — Naam aur Date dono ka size & rang alag set karein. Text kabhi image se bahar nahi jaata (auto-fit).
  • Live Preview — har change turant screen par dikhta hai, download se pehle.
  • Photo clear karein — Brightness, Contrast, Saturation aur Grayscale / Sepia / Warm / Cool / Vintage filters.

Kaise istemaal karein

  1. Apni photo upload karein (JPG / PNG).
  2. "Add Name & Date in Image" section mein apna Naam aur Date type karein.
  3. Zaroorat ho to "Add Extra White Space On Bottom" tick karein.
  4. Font size / colour set karein — preview mein turant dikhega.
  5. Process Karein dabakar edited photo download karein.

100% Secure & Free: Aapki photo sirf aapke browser mein process hoti hai — kisi server par upload nahi hoti. Koi watermark nahi, koi login nahi.
