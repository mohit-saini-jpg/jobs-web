FACT Data Processing Assistant Recruitment 2026 : Frequently Asked Questions (FAQ)

Q1What is the last date to apply for the FACT Data Processing Assistant post?
A
The last date to submit the online application form and send the hard copy of the application along with documents is 28th June 2026.
Q2What is the educational qualification required for this post?
A
Candidates must have completed their Apprenticeship in Computer Engineering from FACT Training & Development Centre and possess a Diploma in Computer Engineering.
Q3What is the age limit for the Data Processing Assistant position?
A
The age limit is between 18 to 35 years, with relaxations applicable for reserved categories as per government norms.
Q4What is the salary offered for the Data Processing Assistant role?
A
Selected candidates will receive a consolidated pay of Rs. 25,000/- per month, with a 3% annual increment.
Q5Is there any application fee for this recruitment?
A
The official notification does not mention any application fee for this recruitment drive.
Q6How will candidates be selected for the Data Processing Assistant post?
A
The selection process involves preliminary document scrutiny, ranking based on apprenticeship details, and a pre-employment medical examination.
Q7What is the mode of application for this recruitment?
A
Applications must be submitted online through the official FACT website, followed by sending the hard copy of the application and documents via post.
Q8What documents need to be uploaded with the online application?
A
Mandatory documents include the application form, X certificate, Diploma certificate, Certificate of Proficiency, Apprenticeship certificate, and caste/disability certificates if applicable.
Q9Where should the hard copy of the application be sent?
A
The original application form and relevant documents should be sent by Speed Post/Registered Post to AGM (HR) Est, HR Department, FEDO Building, FACT, Udyogamandal, PIN 683 501.
Q10Is this a permanent job?
A
No, this is a Data Processing Assistant on Fixed Tenure Contract (Adhoc Basis) position, indicating a contractual appointment.
