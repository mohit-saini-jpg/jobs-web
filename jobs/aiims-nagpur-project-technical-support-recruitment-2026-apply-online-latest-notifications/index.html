Frequently Asked Questions

Q1. What is the last date to apply for the AIIMS Nagpur Project Technical Support III recruitment?

The last date to submit your online application for the Project Technical Support III posts at AIIMS Nagpur is 03 June 2026. It is advisable to apply well before the deadline to avoid any last-minute issues.

Q2. How many vacancies are available for the Project Technical Support III post?

There are a total of 4 vacancies announced for the Project Technical Support III position at AIIMS Nagpur. These positions are on a contractual basis.

Q3. What is the educational qualification required for this post?

Candidates need to have a three-year graduate degree in a relevant subject/field such as Medical Social Work, Sociology, Psychology, or Life Sciences with three years of experience in field-based research. Alternatively, a PG degree in Medical Social Work or Anthropology is also accepted.

Q4. What is the age limit for applying?

The maximum age limit for candidates applying for the Project Technical Support III post is 35 years. Age relaxations are applicable for reserved categories as per government norms.

Q5. Is there any application fee for this recruitment?

No, there is no application fee mentioned for any category (General/OBC, SC/ST) for this recruitment drive. Candidates can apply without paying any fee.

Q6. What is the salary offered for the Project Technical Support III position?

The selected candidates will receive a consolidated salary of Rs. 28,000 per month, along with HRA which is 20% of the basic pay.

Q7. What is the selection process for AIIMS Nagpur Project Technical Support III recruitment?

The selection process for the Project Technical Support III posts will involve a Written Test followed by an Interview. Candidates need to perform well in both stages to be considered for the role.

Q8. What is the mode of application for this job?

The application process for the AIIMS Nagpur Project Technical Support III recruitment is entirely online. Candidates must visit the official website or the provided application link to submit their forms.
