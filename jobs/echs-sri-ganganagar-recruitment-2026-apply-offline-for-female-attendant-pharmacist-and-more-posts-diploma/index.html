Frequently Asked Questions

Q1. What is the last date to apply for the ECHS Sri Ganganagar recruitment?

The last date to submit the offline application for the ECHS Sri Ganganagar recruitment is 06 June 2026. Candidates are advised to submit their applications well in advance of the deadline.

Q2. How many vacancies are available in this ECHS Sri Ganganagar recruitment?

There are a total of 8 vacancies available for various staff positions in ECHS Polyclinic Sri Ganganagar and Hanumangarh as part of this recruitment drive.

Q3. What is the application mode for this recruitment?

The application mode for this ECHS Sri Ganganagar recruitment is offline. Candidates need to download the application form and submit it via speed post.

Q4. What is the highest salary offered in this recruitment?

The highest salary offered for the Officer-in-Charge, Medical Officer, and Dental Officer positions is Rs. 95,000/- per month.

Q5. What is the minimum qualification required for any post?

The minimum qualification required for the Female Attendant post is being literate, and for the Ambulance Driver post, it is an 8th pass with relevant experience.

Q6. Is there any application fee for this recruitment?

No application fee is mentioned in the official notification for any category of candidates applying for this ECHS Sri Ganganagar recruitment.

Q7. What is the duration of the contract for these positions?

The recruitment is for staff on a contractual basis for 11 months, as indicated in the brief description of the advertisement.

Q8. Where should the application be sent?

Applications should be sent in hard copies through speed post to ECHS Cell, Station HQs, Sri Ganganagar Pin 900291 and Civil Pin Code 335001 C/o 56 APO.
